Ccnum
-----

.. note::

   Die ``Ccnum`` Prüfung sollte nicht mehr eingesetzt werden. Stattdessen sollte man die ``CreditCard`` Prüfung
   verwenden. Aus Gründen der Sicherheit sollte man CreditCard statt Ccnum verwenden.
