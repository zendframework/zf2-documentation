Deprecated Validators
---------------------

.. _zend.validator.set.ccnum:

Ccnum
^^^^^

The ``Ccnum`` validator has been deprecated in favor of the ``CreditCard`` validator. For security reasons you
should use CreditCard instead of Ccnum.
