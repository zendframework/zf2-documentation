Panoramica
==========

Il Framework Zend è un framework open source di alta qualità destinato allo sviluppo di applicazioni web e
servizi web in PHP.

Costruito in pieno spirito PHP, il Framework Zend offre funzionalità potenti e facili da utilizzare. Fornisce
soluzioni utili per costruire siti web moderni, robusti e sicuri.
