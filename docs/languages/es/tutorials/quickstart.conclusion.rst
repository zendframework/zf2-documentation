¡Felicitaciones!
================

Ahora ha creado una aplicación muy sencilla utilizando algunas de los componentes mas usados en Zend Framework.
Zend Framework ha creado muchos componentes que se ocupan de la mayoria de lo requerimientos más comunes de las
aplicaciones web incluyendo servicios web,busquedas, *PDF* lectura y escritura, autenticación, autorización,
mucho mas. La :ref:`Guía de Referencia <reference>` es un gran lugar para encontrar más información sobre los
componentes que se ha utilizado en esta guía de inicio rápido, así como otros componentes. Esperamos que
encuentre util Zend Framework y mas importante lo encuentre divertido!
